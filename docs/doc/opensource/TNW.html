简介

TNW: TypeScript(The) + Node.js(Next) + WeChat 微信公众号开发脚手架，支持 http 模块扩展、支持任何 Node.js 的服务端框架(Express、Nest、Egg 等)

安装与运行

NPM 依赖方式

1、下载

$ npm init -y
$ npm i tnw

  • Gitee 请点击这里
  • Github 请点击这里

源码方式

1、下载项目并安装依赖

$ git clone https://github.com/Javen205/TNW.git
或者
$ git clone https://gitee.com/Javen205/TNW.git
$ cd TNW
$ npm install

2、编译并运行

$ npm run build
$ npm run dev
